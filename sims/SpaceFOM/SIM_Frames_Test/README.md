# SIM_Frames_Test


---
### Building the Simulation
In the SIM_Frames_Test directory, type **trick-CP** to build the simulation executable. When it's complete, you should see:

```
Trick Build Process Complete
```

---
### Running the Simulation
In the SIM_Frames_Test directory:

```
./S_main_*.exe RUN_test/input.py --default_data_config --verbose on
```
