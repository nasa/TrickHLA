# SIM_CabinAtmo

SIM_CabinAtmo is a simulation that demostrates the distribute interface.

---
### Building the Simulation
In the SIM_CabinAtmo directory, type **trick-CP** to build the simulation executable. When it's complete, you should see:

```
Trick Build Process Complete
```

---
### Running the Simulation
In the SIM_CabinAtmo directory:

```
./runHLA.sh
```
