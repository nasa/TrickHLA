# SIM_Encoders

---
### Building the Simulation

In the SIM_Encoders directory, type **trick-CP** to build the simulation executable. When it's complete, you should see:

```
Trick Build Process Complete
```

---
### Running the Simulation
In the SIM_Encoders directory:


```
./S_main_*.exe RUN_send/input.py
```

```
./S_main_*.exe RUN_receive/input.py
```
